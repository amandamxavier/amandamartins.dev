## Usage 🔎

- Clone the repository, with the command below:

```console
git clone https://github.com/amandamxavier/amandamartins.dev.git
```

- Install the dependencies:

```console
npm install
```

- Run the project:

```console
npm run dev
```

- ⚠️ Don't forget to run ESLint and Prettier before commit:

```console
npm run lint
```

```console
npm run format
```
